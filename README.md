## Description

`Lilypond` source files for my favorite songs.
Sample outputs are found at https://icy.theslinux.org/muzik/.

## Getting started

1. Use your system package manager to install `lilypond` and `make` tools.
2. Download and install `Choco Cooky` font. You may find the font
   from [this archive][choco-cooky].
3. Now type `make build` to compile all `.ly` files to `.pdf`.

If you want to create your new song, please take a look at the very simple
file [`Sample.ly`](Sample.ly).
